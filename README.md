# lxns_b50

NoneBot2 舞萌 DX 查分插件，集成落雪 (LXNS) 与水鱼 (Diving-Fish) 双数据源 API，提供成绩查询、多维战力构成深度分析、名片自定义生成、今日课题挑战及群内小游戏等功能。

## 功能介绍

### 核心查分
* `b50` / `B50` : 生成 Best 50 个人综合成绩面板。
* `ap50` / `AP50` : 生成 AP 50 纯收曲全成就大图。
* `minfo <曲名/ID>` : 查询单曲在当前玩家处的游玩详情与达成率历史。
* `ginfo <难度><曲名/ID>` : 渲染全服此谱面达成率与 FC 比例统计图表。
* `分数线 <难度><曲名/ID> <达成率目标>` : 计算达成目标率所需的 Perfect/Great/Good/Miss 容错值。

### 玩家个性化
* `dxpass` : 渐进式名片生成助手。允许分步选择卡框等级和名片底图背景，且生成时会自动检测并拉取玩家在落雪平台上当前实际绑定的头像搭档。
* `mai曲线` : 使用 pyecharts 绘制 Rating 历史演变折线走势图（落雪API数据源特供）。
* `mai状态` / `mai个人中心` : 诊断双平台绑定状态与数据同步情况。
* `切换数据源 <水鱼/落雪>` : 实时修改玩家查分默认输出源。
* `mai名片` / `mai信息` : 快速生成一体化街机风格玩家个人属性名片大图。
* `mai排行` / `战力排行` : 渲染全服 TOP 20 战力与装扮大盘。
* `mai分析` / `战力分析` : 多维度分析 Best 50 成绩占比与分布情况。
* `mai速报` / `mai打卡` : 提取最近游玩的优秀成绩生成推特格式打卡图片。
* `mai底力` / `底力` : 高难成绩评级与积分大图。

### 曲目检索与进度
* `查歌 <关键词>` : 模糊检索曲目名。
* `id <曲目ID>` : 调取谱面底标参数与难度详情。
* `定数查歌 <定数>` : 查询对应定数范围的歌曲。
* `bpm查歌 <BPM>` : 查询指定 BPM 的歌曲。
* `<版本><目标>完成表` : 查看特定版本的牌子达成视觉化进度（如霸者完成表、极完成表等）。
* `<版本><目标>进度` / `<等级><评价>进度` : 查看达成目标的完成曲数比例与待完成曲目清单。
* `我要上<分数>` : 根据玩家 Best 50 推荐上分曲目。

### 互动小游戏
* `猜歌` : 开启群内趣味歌词/信息猜曲游戏。
* `猜曲绘` : 开启看局部曲绘大图猜歌游戏。
* `听歌猜歌` : 提取歌曲音频片段在群内播放以猜歌。
* `别名猜歌` : 使用梗别名作为谜题猜官方中文曲名。

## 使用说明

1. 官方机器人支持 Markdown 高级按钮和快捷点击菜单，非官方机器人则自动降级为标准纯文本展示。
2. 双源同步功能会在每日凌晨 04:00 自动执行，获取最新歌曲数据及下载缺失的曲绘资源。
